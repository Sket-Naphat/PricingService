# PricingService

This project is a .NET 8 Core Web API created with the ASP.NET Core Web API template.

## Getting Started

1. Ensure you have .NET 8 SDK installed.
2. To run the project:
   ```powershell
   dotnet run
   ```

## Project Structure
- `Program.cs`: Entry point for the application.
- `Startup.cs` (if present): Configures services and middleware.
- `Controllers/`: Contains API controllers.

## Customization
You can add controllers, services, and other features as needed for your pricing logic.
